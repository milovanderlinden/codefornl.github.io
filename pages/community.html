Community van en voor developers en designers

Samenwerken en ideeën delen

Levende standaarden, best practices, ethische vraagstukken. Sluit aan op een actief netwerk van developers en designers voor uitwisselen van kennis, ervaring en code. In samenwerking met ICTU organiseert Code for NL iedere twee maanden Meetups om met elkaar kennis te maken, ideeën te delen en samenwerkingen aan te gaan.

Internationaal netwerk

Code for NL is aangesloten op internationale netwerken, zoals Code for All en Code for Europe. Jaarlijks zijn er een aantal conferenties waar je vakgenoten kunt ontmoeten van over de hele wereld.

Fellowships

Code for NL organiseert Code Fellowships bij overheden. Gedurende 6 maanden loopt een Code Fellow intern mee om te adviseren op gebied van digitale mogelijkheden en oplossingen te maken voor de problemen die voorbij komen.

Vrijwilligers

Wij zijn een vrijwilligersorganisatie. Ons doel is om te verbinden en te zorgen dat samenwerking tot stand komt. Wij doen dit omdat we het belangrijk vinden in actie te komen. Uiteraard is er meer te doen dan wij aankunnen. Mocht je ons willen helpen, meld je dan aan!
